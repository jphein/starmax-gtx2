# Watch faces (GTX2 dial format)

The Starmax GTX2 (Actions ATS3085x / Runmefit platform) stores a watch face as a packed
container of a background image, a JSON manifest describing element positions, per-digit glyph
fonts, and a preview. These pure-Python tools create, pack, and unpack that container — no
vendor SDK required.

The container layout is documented in [`docs/watchface-format.md`](../docs/watchface-format.md),
and there's an interactive builder page at [`docs/gtx2-face-builder.html`](../docs/gtx2-face-builder.html).

## Tools (`scripts/`)

| Tool | What it does |
|------|--------------|
| `dial_create.py` | Build a valid dial `.bin` from scratch: a background image + a small JSON layout config → manifest + `*_0565.bmp` background + rendered 0–9 glyph fonts + preview, packed via `dial_pack.py`. |
| `dial_pack.py`   | Pack a staged slot directory into a distributable `.bin` (DEFLATE-9). |
| `dial_unpack.py` | Unpack a `.bin` back into its manifest + assets, for inspection. |

```sh
# inspect the format
python scripts/dial_unpack.py custom/sample_clock.bin --outdir /tmp/dial

# build your own (see the config schema in scripts/dial_create.py's docstring)
python scripts/dial_create.py my_layout.json my_dial.bin
```

`dial_create.py` renders digit glyphs with a TrueType font. It looks for DejaVu / Liberation in
the usual system locations; override with **`$GTX2_FONT_DIR`** pointing at a directory that
contains `DejaVuSans-Bold.ttf` (or `LiberationSans-Bold.ttf`). Requires Pillow.

Uploading a face to the watch is done by the BLE client (`client/`): `starmax dial-push <bin>`.

## Sample

`custom/sample_clock.bin` is a dial generated by `dial_create.py` — a working reference you can
unpack to see the exact container layout.

> **Vendor stock dials are not distributed here.** The factory dials extracted from the watch or
> the vendor's dial store are the vendor's copyrighted artwork; only original, tool-generated
> faces are committed. Build your own with `dial_create.py`.
